TypeScript Operating System

Fall 2024

Operating Systems class project at Marist Universirty with Dr. Labouseur

Developed an operating system in TypeScript that ran multiple 6502-instruction programs using a round-robin CPU scheduler. Behind the scenes, the operating system includes a kernel interrupts with interupts, Memory, an MMU, and a Disk with swap file capability and a user file system. The OS contains a command line with several commands that allow users to input, run, and check progress of programs. Program output is also put here. I innovated the project by implementing extra shell features like text class — adding multiple colors, spacing, and tags. Featured in Dr. Labouseur's Hall of Fame
